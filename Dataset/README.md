The dataset used in this project is sourced from Kaggle.

Dataset: Brain Tumor MRI Dataset
Link: [https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset](https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset)

The dataset contains labeled MRI images of four distinct categories: Glioma, Meningioma, Pituitary tumors, and No Tumor cases.
It is designed to support the training and evaluation of deep learning models for automated medical image diagnosis and tumor detection.
The objective is to accurately classify each input MRI scan into one of these four predefined clinical categories to assist in medical diagnosis.
